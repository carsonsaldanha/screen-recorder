Hey! Thanks for using this screen recorder!

  • This web app is completely free to use and open source with no ads at all. It was made using pure HTML, CSS, and JavaScript, without any external frameworks.
    • Click here to view the source code on GitHub.
  • It's private. There are no analytics, trackers, or cookies. All recordings are processed locally, on-device. No data is sent to a server.
  • It's also a Progressive Web App (PWA)! This means you can "install" it on your device for easy access and to use when offline. It works on Chrome OS, macOS, Windows, and Linux.
    • Click here to learn how to install a PWA in Chrome.
  • This web app uses modern web APIs and technologies, so if you're running into any issues, please use the latest versions of Google Chrome, Mozilla Firefox, Microsoft Edge, Apple Safari, or Opera.

This was my first time doing web development of any kind! If you found this app handy, you can support the creator through a PayPal donation here.
